================================
 LGU Record Keeping System
 Portable EXE Version
================================

QUICK START:
1. Double-click "LGU_Record_System.exe"
2. Wait for the console window to show "Running on http://127.0.0.1:5000"
3. Open your web browser to: http://localhost:5000
4. Login with: admin / admin123

DEFAULT USERS:
- Admin: admin / admin123
- User:  user / user123

IMPORTANT NOTES:
- This is a PORTABLE version - no installation required!
- No Python installation needed on target computer
- All data files are created automatically
- Keep the EXE file in its own folder for best results
- Your data is saved in JSON files in the same folder

SYSTEM REQUIREMENTS:
- Windows 7/8/10/11
- 50MB free disk space
- Any web browser
- No other software required!

NETWORK ACCESS:
To allow other computers to access this system:
1. Make sure Windows Firewall allows the program
2. Other computers can access via: http://YOUR_IP:5000
3. Find your IP with: ipconfig

TROUBLESHOOTING:
- If antivirus blocks it: Add to exclusions (this is normal for EXE files)
- If it won't start: Run as Administrator
- If port 5000 is busy: Close other programs or restart computer
- For network access: Check Windows Firewall settings

DATA LOCATION:
All your records, users, and settings are saved in the same folder
as the EXE file. These files are created automatically:
- records.json (your data)
- users.json (user accounts)
- system_settings.json (configuration)

UPDATES:
To update, simply replace the EXE file with a newer version.
Your data files will be preserved.
